八宫筮法与中医诊断 ​

京氏易的"世应飞伏"在辨证中的运用 从临床角度来看"卦象断病"

一、世应——病位与病证的对应 ​

京房八宫每卦都有"世爻"和"应爻"：

宫世位病位对应世爻含义
八纯（本宫）上爻世久病、先天、老病病根最深
一世初爻世新病、表证病在皮毛经络
二世二爻世病在肌肉脾胃之病
三世三爻世病在脏腑五脏之病
四世四爻世病在骨髓深重难愈
五世五爻世病在神明精神之病
游魂四爻游魂游走不定痛无定处
归魂三爻归魂回归本位恢复期

临床用法：

  • 多用于疑难病诊断——当常规辨证无效时，可参考世应断病位

二、飞伏——隐证与显证 ​

飞 = 显现的卦象（表面症状） 伏 = 隐藏的卦象（根本病机）

飞伏辨证法 ​

飞（表面）伏（根本）临床意义
离火（上热）坎水（下寒）上热下寒（患者（男）·患者（女））
震木（肝郁）坤土（脾虚）木郁克土
兑金（肺热）艮土（胃寒）上热中寒
巽风（疏泄）坎水（肾封藏）疏泄太过→肾不藏精

患者的飞伏辨证：

  • 飞（显）：离宫未济䷿——口渴、前胸出汗、血压高（上热）
  • 伏（藏）：坎宫坎䷜——腰凉、脚冷（下寒）

治法：见飞（上热）不能治飞——因为飞是表面的，伏才是根本。

  • 如果见飞治飞（用清热药）→ 使伏更寒 → 上热更重
  • 正确的治法：治伏调飞 → 温坎水 → 离火自降

三、六爻与人体对应 ​

爻位人体部位患者症状
上爻（天）头·巅顶头顶疤近百会、头发花白
五爻（天）面·目·鼻面色暗沉、目干涩、鼻疮
四爻（人）胸·背前胸出汗多（关键）
三爻（人）腹·胃腰部赘肉（关键）
二爻（地）腰·少腹腰凉（核心）
初爻（地）足·胫脚后跟裂、脚气

患者（男）六爻诊断：

  • 上焦（四·五·上）= 离火浮越（前胸出汗+口渴+血压高）→ 三爻阳盛
  • 下焦（初·二·三）= 坎水寒（腰凉+脚冷）→ 二爻阴盛
  • 关键在第三爻（人位·腹）：腰凉 + 腹部赘肉 → 中焦枢轴不转

四、用于患者的完整筮法诊断 ​

得卦：未济䷿ → 既济䷾ ​

未济䷿（当前状态）：

上卦：离☲（火）→ 相火不降 → 前胸出汗+口渴+血压高
下卦：坎☵（水）→ 肾水寒 → 腰凉+脚冷

既济䷾（治疗目标）：

上卦：坎☵（水）→ 水在上 → 津液能上达 → 口不渴
下卦：离☲（火）→ 火在下 → 下焦温暖 → 腰不凉

如何从未济到既济：

  • 患者方中的牛膝引火下行 = 从未济的"火在上"→ 既济的"火在下"
  • 患者方中的附子温肾水 = 从未济的"水在下"→ 既济的"水润上"

五、小结 ​

八宫筮法用于临床的要点：

1. 见到"飞"（表面症状），不要治"飞"——要治"伏"（根本病机）。2. 上热下寒 = 未济䷿ → 目标是既济䷾。3. 八宫定位可以精准判断病位（初爻=足、二爻=腰、三爻=腹……）。4. 世应判断病程长短。5. 归魂=恢复期、预后良好。

医学层面的京氏易应用已全部涵盖于此。
