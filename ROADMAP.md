## 一、本周冲刺（6月7日-13日）：社交裂变 + 基础变现

### 1.1 分享卡片生成 ← 最高优先级
- 用 Canvas 生成精美报告分享图
- 包含：八字四柱、能量曲线缩略图、日主五行、命局简评
- 自动适配微信/朋友圈/小红书尺寸
- 二维码自动生成（引导扫码）

### 1.2 免费 → 付费分级
- **免费版**：八字神卡 + 能量K线 + 报告前2个模块（性格+优势弱点）
- **付费版(9.9元)**：完整6模块 + PDF导出 + 流年详批
- 支付接入：微信支付/支付宝（使用Vercel Edge Functions或第三方）

### 1.3 流年详批功能
- 单独付费功能（4.9元/次）
- 输入年份 → AI生成该年度详细指引
- 包含：流年八字、五行冲克、每月吉凶、决策建议

### 1.4 技术准备
- 接入微信支付/支付宝（推荐使用 `payjs.cn` 或 `stripe` 出海版）
- 存储升级：从 localStorage 改为云端（可选，初期可先用 localStorage）

- 翻译UI + prompt 为英文
- 适配国际占星体系（Western Astrology + Numerology）
