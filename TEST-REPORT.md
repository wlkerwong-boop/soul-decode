## 三、关键功能测试

### 3.1 用户流程测试

```
首页 → 八字输入 → 免费八字排盘（八字卡片+能量K线）
  ↓
AI报告流式生成（所有用户可见）
  ↓
AI追问对话（3次免费）
  ↓
每日运势页面
  ↓
登录/注册 → 个人档案（历史报告）
  ↓
支付 → 会员解锁
```

### 3.2 法藏RAG测试

- 索引规模：437个文件、630万字、2394个内容块、256,576个关键词
- 搜索流程：问题输入 → 关键词匹配 → 取Top5相关块 → AI生成回答
- 引用来源：每条回答标注来自哪些开示文件
- 构建命令：`npm run index:dharma`
